Dynamic google sitemap.xml generator

== Description ==

Select what post types you want to appear in your sitemap.xml file and they are dynamically generated at /sitemap.xml.  

== Installation ==

1. Upload `tj-sitemap` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Tools->TJ Sitemaps and select the post types that you would like to appear and it will take care of the rest.

== Frequently asked questions ==

= Its not working, why? =

Please check to make sure that:
* You don't have any other sitemap plugins active
* There is no file in existence called sitemap.xml in the root of your Wordpress installation
